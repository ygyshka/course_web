# Разработка сервиса

## Проект проектирования программы ервис управления рассылками, администрирования и получения статистики..

    """ Чтобы удержать текущих клиентов, часто используют вспомогательные, или «прогревающие», рассылки для информирования и привлечения клиентов.
        вашему вниманию представлен проект сервиса управления рассылками, администрирования и получения статистики. """

## Основные шаги проекта

- Реализовать интерфейс заполнения рассылок, то есть CRUD-механизм для управления рассылками`.
- Реализовать скрипт рассылки, который работает как из командой строки, так и по расписанию.
- Внести настройки конфигурации для периодического запуска задачи.
- Расширить модель пользователя для регистрации по почте, а также верификации.
- Добавить интерфейс для входа, регистрации и подтверждения почтового ящика.
- Создать блог для продвижения сервис.
